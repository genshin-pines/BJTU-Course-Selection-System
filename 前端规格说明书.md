## 4. Axios 封装 — `utils/request.js`

```javascript
const request = axios.create({
  baseURL: '/api',          // 所有接口前缀 /api
  timeout: 30000            // 30 秒超时
})

// 请求拦截：自动注入 Bearer token
request.interceptors.request.use(config => {
  const token = sessionStorage.getItem('token')
  if (token) config.headers.Authorization = `Bearer ${token}`
  return config
})

// 响应拦截：
//  - code !== 200 → ElMessage.error + Promise.reject
//  - 401 → 清 session + 跳转登录页（根据 role 区分学生/管理员）
//  - 403 → ElMessage.error("无权限访问")
//  - skipAuthRedirect → 阻止 401 时自动跳转
//  - silentError → 不弹 ElMessage
```

**关键 config 标志**：
| 属性 | 作用 |
|------|------|
| `skipAuthRedirect: true` | 401 时不自动跳转登录页（session 验证用） |
| `silentError: true` | 403/业务错误不弹 ElMessage（点赞点踩用） |
| `responseType: 'blob'` | 下载文件（导入模板） |

## 5. Pinia Store — `stores/auth.js`

### State

```javascript
{
  token:       '',           // JWT token（sessionStorage 持久化）
  userInfo:    null,         // { anonymousId, studentNo, name } 或 { username }
  userRole:    '',           // 'STUDENT' | 'ADMIN' | ''
  adminRole:   '',           // 'SUPER_ADMIN' | 'DEPT_OP' | 'AUDITOR'
  sessionChecked: false      // 是否已完成 session 验证
}
```

### Getters

```javascript
isLoggedIn  → !!token
isStudent   → !!token && userRole === 'STUDENT'
isAdmin     → !!token && userRole === 'ADMIN'
```

### Actions

| 方法 | API | 说明 |
|------|-----|------|
| `studentLogin(studentNo, password)` | POST `/auth/login` | 存 token + userInfo，设 role='STUDENT' |
| `studentRegister(email, studentNo, password, code)` | POST `/auth/register` | 仅调接口，不存状态 |
| `sendCode(email, studentNo)` | POST `/auth/send-code` | 发送验证码 |
| `adminLogin(username, password)` | POST `/auth/admin/login` | 存 token + adminRole + userInfo，设 role='ADMIN' |
| `verifySession(force)` | GET `/auth/session` | 带 skipAuthRedirect，验证后写回 sessionStorage |
| `syncFromStorage()` | — | 从 sessionStorage 恢复 state（不调接口） |
| `logout()` | — | 清空所有 state + sessionStorage |

### 持久化策略

```
token      → sessionStorage.getItem('token')
userRole   → sessionStorage.getItem('userRole')
adminRole  → sessionStorage.getItem('adminRole')
userInfo   → sessionStorage.getItem('userInfo') (JSON)
```

```javascript
router.beforeEach(async (to, from, next) => {
  // 1. 设置 document.title
  // 2. 判断是否需要鉴权（requiresStudent / requiresAdmin）
  //    - 需要鉴权 → await authStore.verifySession(true)
  //    - 不需要 → authStore.syncFromStorage()
  // 3. requiresStudent && !isStudent → next('/login')
  // 4. requiresAdmin && !isAdmin → next('/admin/login')
  // 5. 放行
})
```

**数据流**:
| 操作 | API | 成功回调 |
|------|-----|----------|

- `studentNo`: required, 8 位数字
- `password`: required, minLength 6
- `email`: required, 邮箱格式, 必须以 @bjtu.edu.cn 结尾
- `confirmPassword`: required, 与 password 一致
- `code`: required, 6 位

| 点踩 | `reviewApi.downvote(id)` | silentError |
